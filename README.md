# Simple Chatbot GUI

A small rule-based chatbot built with Python and Tkinter. It matches user input against a set of keyword rules (greetings, farewells, small talk) and replies with a randomized response inside a simple, dark-themed chat window.

This was built as a beginner project to practice GUI programming and event-driven logic in Python.

## Features

- Dark-themed chat window built with Tkinter
- Keyword-based response matching (greetings, "how are you?", "bye", etc.)
- Randomized replies for more natural-feeling responses
- Send messages by clicking the button or pressing Enter

## Tech Stack

- Python 3
- Tkinter (standard library, no extra installs needed)

## How to Run

1. Make sure Python 3 is installed.
2. Clone this repository:
   ```bash
   git clone https://github.com/sangothayan-y/simple-chatbot-gui.git
   cd simple-chatbot-gui
   ```
3. Run the chatbot:
   ```bash
   python chatbot.py
   ```

## Example

```
you: hi
bot: Hello

you: how are you?
bot: I'm good

you: bye
bot: see you later!
```
